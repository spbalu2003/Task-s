What is Browser?

A browser (short for web browser) is a software application used to access and view websites on the internet.

Example:Google chrome,firefox,opener etc..

What is internet?

internet is a connection of no.of computers to share the info.

example: sending mails,online games

what is Domain?

A domain is the name of a website in a URL. It helps people find websites easily.

example: google.com, whatsapp.com, etc...

What is URL?

URL (Uniform Resource Locator) is the address of a webpage. It tells the browser where to go.

example: https://www.google.com, https://www.whatsapp.com/login etc..

Difference B/W website & webapp ?

A website mainly gives information and is mostly static. Examples include Wikipedia, BBC News, and personal blogs.

A web application is interactive and lets users perform tasks. Examples include Gmail, Facebook, and Amazon.

In short: Websites show info; web apps let you do things.
